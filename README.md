## 数値
### integer
- 整数
- 大きい数値はアンダースコアで区切れる
  - 例：```1_000_000```
- 16, 8, 2 進数表記ができる
  - 0x
  - 0o
  - 0b

### float
- 浮遊動小数点数
- e: 指数
  - 指数が0の場合は省略可能
